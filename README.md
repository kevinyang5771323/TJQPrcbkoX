# 应急预案管理系统 java844

## 项目概述

应急预案管理系统是一款面向大型商场的智能化管理系统，旨在提高商场应对各类紧急事件的效率和协调性。系统基于 Java 和 SpringBoot 技术栈开发，采用前后端分离的设计模式，前端使用 Vue 框架搭建，后端采用 SSM 框架，数据库使用 MySQL。

## 技术栈

- 后端：Java，Spring Boot（SSM）
- 前端：Vue（ElementUI + JavaScript + CSS）
- 数据库：MySQL
- 项目构建：Maven

## 功能模块

### 系统角色

- 员工
- 管理员

### 员工功能

- 登录
- 修改个人信息
- 查看预案信息
- 查看对应预案
- 查看类型统计

### 管理员功能

- 登录
- 修改个人信息
- 员工管理
- 预案信息管理
- 添加对应预案
- 预案类型管理
- 事件类型管理
- 数据统计（按预案类型，按事件类型）
- 对应预案管理

## 运行环境

- MySQL（Navicat）
- Maven

## 部署步骤

1. 准备所需运行环境（JDK、IDEA、MySQL、Maven）。
2. 克隆或下载项目源代码。
3. 部署数据库，执行数据库初始化脚本。
4. 使用 Maven 对项目进行编译和打包。
5. 部署应用到服务器。
